# Landing Page eter

Descripcion
Landing page de eter con secciones de producto, beneficios, precios, testimonios, blog y legales. La app esta hecha con React y Vite, usando React Router para la navegacion interna.

Guia rapida para comenzar

1. Instala dependencias:
   npm install

2. Inicia el entorno de desarrollo:
   npm run dev
