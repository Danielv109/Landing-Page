# Landing Page eter

Este proyecto corresponde a la landing page de eter, una plataforma SaaS modular orientada a industrias de salud, restaurantes, retail y servicios profesionales. La pagina comunica la propuesta de valor, describe el flujo funcional del producto y presenta los modulos disponibles con un enfoque comercial y operativo. Incluye secciones de beneficios, comparativas y llamados a la accion, junto con planes de precios y testimonios para respaldar la credibilidad de la oferta.

El sitio incorpora un formulario de captura de leads para iniciar una prueba, con validacion de datos y envio mediante integracion con EmailJS. Asimismo, contiene contenido informativo adicional mediante un blog, con paginas de listado y detalle por articulo. Se incluyen secciones legales completas (terminos de servicio, aviso de privacidad, politica de cookies y aviso legal) para cumplimiento y comunicacion institucional.

La aplicacion esta implementada con React y Vite, utilizando enrutamiento interno para las vistas principales y navegacion sin recarga entre secciones. El resultado es una interfaz de una sola pagina con rutas dedicadas para contenido legal y blog, manteniendo consistencia visual y estructura modular. La base del proyecto prioriza rendimiento, escalabilidad del contenido y una arquitectura de componentes reutilizables.
